<en-US>
🐛 Leaderboard / ads silently broken — fixed

Root cause
• Since v0.24.0, R8 code shrinker was stripping the PGS and AdMob plugin
  classes, so sign-in, score submission and ads all silently no-op'd.

Fix
• proguard rules now explicitly -keep both plugin namespaces.
• Both classes load correctly starting this build.
