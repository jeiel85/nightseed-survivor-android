# STORY_UI_COPY_NIGHTSEED_PATCH

이 문서는 Nightseed 의미를 UI 문구와 스토어 문구에 반영하기 위한 카피 가이드다.

## 1. 로딩 화면 문구

### Forest of Echoes

```text
밤의 씨앗이 처음 숨을 쉬던 숲
```

### Frozen Wastes

```text
잊힌 맹세가 얼어붙은 땅
```

### Twilight Sanctum

```text
기억을 잠가 둔 황혼의 방
```

### Inferno Chasm

```text
태울 수 없는 씨앗을 비추는 불길
```

### Cursed Tomb

```text
이름이 봉인된 기록의 무덤
```

---

## 2. 튜토리얼 또는 첫 실행 힌트

```text
밤의 씨앗이 자라날수록 적은 강해집니다.
살아남아 기억의 조각을 되찾으세요.
```

영어:

```text
As the Nightseed grows, enemies become stronger.
Survive and recover fragments of memory.
```

---

## 3. 보스 경고 문구

```text
봉인의 파수꾼이 깨어납니다.
```

영어:

```text
A warden of the seal awakens.
```

---

## 4. 클리어 화면 보조 문구

```text
기억의 조각을 되찾았습니다.
```

영어:

```text
A fragment of memory has returned.
```

## 6. 스토어 문구

### 짧은 한국어 소개

```text
밤의 씨앗이 깨어난 세계.
이름을 잃은 기사는 달빛이 기억하는 진실을 따라간다.
```

### 긴 한국어 소개

```text
Nightseed Survivor는 자동 공격과 성장 선택으로 몰려오는 적을 버티는 2D 생존 액션 게임입니다.

하지만 밤의 씨앗은 단순한 어둠이 아닙니다.
그것은 잊힌 맹세와 기억이 뭉쳐 생긴 봉인의 핵입니다.

스테이지를 넘을수록,
당신의 검은 잃어버린 이름과 봉인의 진실에 가까워집니다.
```

### 짧은 영어 소개

```text
The Nightseed awakens.
A nameless knight follows the truth remembered by moonlight.
```

### 긴 영어 소개

```text
Nightseed Survivor is a 2D survival action game built around auto-firing weapons and quick upgrade choices.

But the Nightseed is not darkness itself.
It is the sealed core of forgotten oaths and memories.

Survive each stage, defeat the wardens,
and uncover the truth your blade still remembers.
```
