## Nightseed 용어 정의

### 기본 의미

```text
Nightseed = 밤의 씨앗
```

Nightseed는 세계에 밤을 자라나게 만드는 봉인된 기억의 씨앗이다.

어둠 그 자체도 아니고, 숭배하거나 계약하는 존재도 아니다.

그것은 오래전 세계를 지키기 위해 봉인된 기억과 맹세의 핵이다.

봉인이 약해지면 Nightseed는 밤처럼 번지고, 사람과 장소는 자신의 이름과 기억을 잃기 시작한다.

### 스토리 기능

Nightseed는 게임 전체의 중심 미스터리다.

처음에는 어둠의 원인처럼 보인다.

하지만 스테이지를 진행할수록 단순히 파괴해야 할 악이 아니라, Vagrant의 과거와 연결된 봉인의 핵이라는 사실이 드러난다.

### 플레이 구조와의 연결

| 게임 요소 | Nightseed 의미 |
|---|---|
| 시간이 지날수록 적이 강해짐 | Nightseed가 점점 자라남 |
| 각 스테이지의 적 구성 차이 | 지역마다 다른 기억 왜곡 |
| 미니보스 | 봉인의 잔향 또는 기억의 파편 |
| 최종 보스 | 봉인을 지키던 파수꾼 |
| Vagrant의 검 | Nightseed 봉인을 닫았던 열쇠 |
| 검의 균열 | 상처가 아니라 봉인 지도의 조각 |

### 한 줄 정의

```text
Nightseed는 세계에 밤을 자라나게 만드는 봉인된 기억의 씨앗이다.
```

### 시적 정의

```text
밤의 씨앗은 어둠이 아니라, 잊힌 맹세가 자라난 흔적이다.
```

### 5차 복선

Nightseed는 적의 근원이지만, 완전히 악한 존재는 아니다.

그것은 잘못된 기억과 깨진 맹세가 봉인 안에서 자라난 결과다.

### 최종 반전 보강

Vagrant는 Nightseed를 단순히 없애야 하는 사람이 아니라, Nightseed를 봉인하기 위해 자신의 이름을 대가로 바친 마지막 기사였다.
