# Story UI Copy

## 1. 목적

이 문서는 Nightseed 세계관을 UI, 로딩 문구, 스테이지 설명, 스토어 문구에 짧게 반영하기 위한 카피 기준이다.

## 2. 로딩 화면 문구

| 스테이지 | 한국어 | 영어 |
|---|---|---|
| Forest of Echoes | 밤의 씨앗이 처음 숨을 쉬던 숲 | The forest where the Nightseed first breathed. |
| Frozen Wastes | 잊힌 맹세가 얼어붙은 땅 | A land where forgotten oaths froze over. |
| Twilight Sanctum | 기억을 잠가 둔 황혼의 방 | A twilight chamber where memories are sealed. |
| Inferno Chasm | 태울 수 없는 씨앗을 비추는 불길 | Flames that reveal the seed that cannot burn. |
| Cursed Tomb | 이름이 봉인된 기록의 무덤 | A tomb of records where a name was sealed. |

## 3. 튜토리얼 또는 첫 실행 힌트

한국어:

```text
밤의 씨앗이 자라날수록 적은 강해집니다.
살아남아 기억의 조각을 되찾으세요.
```

영어:

```text
As the Nightseed grows, enemies become stronger.
Survive and recover fragments of memory.
```

## 4. 보스 경고 문구

한국어:

```text
봉인의 파수꾼이 깨어납니다.
```

영어:

```text
A warden of the seal awakens.
```

## 5. 클리어 화면 보조 문구

한국어:

```text
기억의 조각을 되찾았습니다.
```

영어:

```text
A fragment of memory has returned.
```

## 6. 스테이지 선택 설명

| 스테이지 | 한국어 | 영어 |
|---|---|---|
| Forest of Echoes | 달빛이 잃어버린 이름을 부르는 숲. | A forest where moonlight calls a lost name. |
| Frozen Wastes | 깨어진 맹세가 얼음 아래 잠든 땅. | A frozen land where broken oaths sleep below. |
| Twilight Sanctum | 기억을 봉인한 황혼의 방. | A twilight chamber that sealed memories away. |
| Inferno Chasm | 씨앗을 태우는 대신 비춰야 하는 심연. | A chasm where the seed must be revealed, not burned. |
| Cursed Tomb | 마지막 기사의 이름이 기록된 무덤. | A tomb that records the last knight's name. |

## 7. 스토어 문구

짧은 한국어 소개:

```text
밤의 씨앗이 깨어난 세계.
이름을 잃은 기사는 달빛이 기억하는 진실을 따라간다.
```

긴 한국어 소개:

```text
Nightseed Survivor는 자동 공격과 성장 선택으로 몰려오는 적을 버티는 2D 생존 액션 게임입니다.

하지만 밤의 씨앗은 단순한 어둠이 아닙니다.
그것은 잊힌 맹세와 기억이 뭉쳐 생긴 봉인의 핵입니다.

스테이지를 넘을수록,
당신의 검은 잃어버린 이름과 봉인의 진실에 가까워집니다.
```

짧은 영어 소개:

```text
The Nightseed awakens.
A nameless knight follows the truth remembered by moonlight.
```

긴 영어 소개:

```text
Nightseed Survivor is a 2D survival action game built around auto-firing weapons and quick upgrade choices.

But the Nightseed is not darkness itself.
It is the sealed core of forgotten oaths and memories.

Survive each stage, defeat the wardens,
and uncover the truth your blade still remembers.
```
