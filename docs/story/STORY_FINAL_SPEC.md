# Story Final Spec

## 1. 목적

이 문서는 Nightseed Survivor의 스토리 기반 구현을 위한 최종 기준이다.

현재 단계의 목표는 복잡한 컷신을 추가하는 것이 아니라, 게임명과 스테이지, 적, 보스, UI 문구가 같은 세계관을 공유하도록 기준을 고정하는 것이다.

## 2. 핵심 전제

- 플레이어 캐릭터 Vagrant는 자신의 이름과 과거를 잃은 방랑자다.
- 세계는 Nightseed가 깨어나며 밤과 망각에 잠식되고 있다.
- 각 스테이지는 서로 다른 기억 왜곡을 보여준다.
- 보스는 단순한 악당이 아니라 봉인과 기억에 묶인 파수꾼이다.
- Vagrant의 검은 봉인을 닫았던 열쇠이자 잃어버린 기억의 지도다.

## 3. Nightseed 용어 정의

```text
Nightseed = 밤의 씨앗
```

Nightseed는 세계에 밤을 자라나게 만드는 봉인된 기억의 씨앗이다.

어둠 그 자체도 아니고, 숭배하거나 계약하는 존재도 아니다. 오래전 세계를 지키기 위해 봉인된 기억과 맹세의 핵이다.

봉인이 약해지면 Nightseed는 밤처럼 번지고, 사람과 장소는 자신의 이름과 기억을 잃기 시작한다.

한 줄 정의:

```text
Nightseed는 세계에 밤을 자라나게 만드는 봉인된 기억의 씨앗이다.
```

시적 정의:

```text
밤의 씨앗은 어둠이 아니라, 잊힌 맹세가 자라난 흔적이다.
```

## 4. 메인 스토리 요약

Nightseed는 어둠 그 자체가 아니다.

그것은 오래전 닫힌 봉인의 핵이며, 잊힌 이름과 맹세가 씨앗처럼 뭉쳐진 것이다.

그 씨앗이 깨어날 때 세계는 밤에 잠식되고, 사람들은 자신의 이름을 잃는다.

Vagrant가 잃어버린 이름 역시 그 봉인의 대가였다.

플레이어는 스테이지를 진행하며 Nightseed가 단순히 파괴해야 할 악이 아니라, Vagrant의 과거와 연결된 봉인의 핵이라는 사실을 알아간다.

## 5. 플레이 구조와의 연결

| 게임 요소 | 스토리 의미 |
|---|---|
| 시간이 지날수록 적이 강해짐 | Nightseed가 점점 자라남 |
| 각 스테이지의 적 구성 차이 | 지역마다 다른 기억 왜곡 |
| 미니보스 | 봉인의 잔향 또는 기억의 파편 |
| 최종 보스 | 봉인을 지키던 파수꾼 |
| Vagrant의 검 | Nightseed 봉인을 닫았던 열쇠 |
| 검의 균열 | 상처가 아니라 봉인 지도의 조각 |
| 골드와 영구 강화 | 반복 도전 속에서 회복되는 의지와 기억 |

## 6. 반전 구조

### 초기 인식

Nightseed는 세계를 어둡게 만드는 원인처럼 보인다.

### 중반 단서

Nightseed는 맹세와 봉인, 잊힌 이름과 연결되어 있음이 드러난다.

### 후반 반전

Nightseed는 적의 근원이지만 완전히 악한 존재는 아니다. 잘못된 기억과 깨진 맹세가 봉인 안에서 자라난 결과다.

Vagrant는 Nightseed를 단순히 없애야 하는 사람이 아니라, Nightseed를 봉인하기 위해 자신의 이름을 대가로 바친 마지막 기사였다.

### 후속 떡밥

밤의 씨앗은 아직 하나가 아니다.

## 7. MVP 적용 범위

MVP에서는 아래까지만 적용한다.

- 스테이지 설명 문구에 세계관 반영
- 보스 경고/클리어 보조 문구 기준 수립
- 향후 용어집 UI를 위한 `story_terms.json` 추가
- 복잡한 컷신, 긴 대화, 분기 스토리 구현은 보류

## 8. 금지 표현

Nightseed는 아래 요소와 연결하지 않는다.

- 악마의 씨앗
- 제물 의식
- 피의 봉인
- 숭배
- 계약
- 타락한 신성
- 실제 종교 조롱 또는 모사
