## 🔎 v0.23.0 — 폰트 대확대 (모바일 가독성 최종 보정)

### 🎯 핵심 변경

v0.22.0의 폰트 +4 폭이 모바일에서 여전히 "답답하다"는 피드백 받고, 이번엔
**메인 메뉴와 HUD의 모든 폰트를 ~1.7배 수준으로 재조정**. 게임플레이는 손대지 않음.

---

### 📱 메인 메뉴 — 모든 글자 크게

| 위치 | v0.22 → v0.23 |
|---|---|
| 타이틀 (NIGHTSEED SURVIVOR) | 58 → 76 |
| 부제 (5분 생존 액션) | 24 → 36 |
| 골드 카드 | 30 → 46 |
| 상태 라인 (방랑자 · 메아리의 숲 · 일반) | 20 → 30 |
| PLAY 버튼 | 48 → 76 |
| 1차 / 2차 버튼 (캐릭터/스테이지/난이도/상점/스토리/순위표) | 22 → 36 |
| 우상단 코너 (언어/크레딧) | 17 → 24 |

컨테이너 높이도 같이 키워서(PLAY 136→196, 1차 행 88→128, 2차 행 80→120) 폰트가 잘리지 않게.

---

### ❤ HUD — 한눈에 들어오게

| 위치 | v0.22 → v0.23 |
|---|---|
| 시간 (4:19 등) | 34 → 56 |
| Lv / Kills / Gold 라벨 | 20 → 34 |
| HP 라벨 | 18 → 28 |
| 스탯 아이콘 | 26×26 → 40×40 |
| HP 바 두께 | 28 → 42 |
| top bar 전체 높이 | 140 → 210 |

HP 라벨 외곽선도 4→5, 알파 0.75→0.85로 강화 — 큰 폰트에 맞춰 HP 바 어떤 fill 색이든 잘 읽히게.

레벨업 카드는 v0.22에서 이미 충분히 크다고 판단해 손대지 않음.
