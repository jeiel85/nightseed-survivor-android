## 🌙 v0.28.1 — 레벨업 카드 표시 버그 수정

### 🎯 핵심 변경

레벨업 화면에서 **정령의 구**의 CD가 `4575.54초 → 4026.48초` 같은 비현실적인 숫자로 표시되고, 그 텍스트가 카드 폭을 그대로 넘어 화면 밖으로 잘리던 문제를 수정했습니다.

---

### 🐛 정령의 구 CD 표시 버그

정령의 구는 다른 무기와 달리 발사 타이머(`base_cooldown`)가 아니라 별도의 `DAMAGE_INTERVAL = 0.35초` 타이머로 데미지를 처리합니다. 그래서 `base_cooldown` 값이 `9999`로 의도적으로 큰 sentinel 값으로 박혀 있었는데, 레벨업 카드의 stats 표기는 모든 무기 공통으로 `base_cooldown × cooldown_multiplier` 를 그대로 출력하고 있었습니다. 결과적으로 `9999 * 0.45 ≈ 4575` 같은 의미 없는 숫자가 노출됐습니다.

수정:

- `WeaponBase`에 `get_display_cooldown()` / `get_display_next_cooldown()` 가상 메서드 추가. 기본 구현은 기존 계산 그대로.
- `SpiritOrb`가 두 메서드를 override 하여 `DAMAGE_INTERVAL × cooldown_multiplier` 를 반환. 업그레이드는 구체 수를 늘리는 방식이라 next == current.
- `LevelUpUI._generate_options()` 가 base_cooldown 직접 곱셈 대신 새 메서드를 호출.

---

### 🛡️ 카드 텍스트 오버플로 안전망

`LevelUpUI.tscn` 의 Card1/Card2/Card3 모두 `Stats` Label 에 `autowrap_mode = 3` (WORD_SMART) 를 추가. 향후 어떤 사유로든 stats 문자열이 길어져도 카드 폭을 넘어 잘리는 일이 없도록 보호.
