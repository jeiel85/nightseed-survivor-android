## v0.36.0 — 난이도 리워크와 화면 통일

### 핵심 변경

"거의 자동사냥 가능" 수준이던 난이도를 봇 측정 기반으로 전면 재조정하고, 화면별로 제각각이던 하위 UI(상점/캐릭터/스테이지/설정)를 단일 테마로 통일했습니다.

### Changed — 난이도 리워크 (docs/BALANCE.md §12)

- **적 재배치 신규**: 한 방향 도주 시 적 수 상한 도달 → 신규 스폰 정지 → 무리가 꼬리에만 끌려오는 circle-kiting 무한 생존 루트를 차단. 플레이어에게서 1050px 이상 떨어진 적은 진행 방향 앞쪽 스폰 링으로 재배치됩니다 (보스/미니보스 제외).
- Normal 배수 0.85/0.9 → 1.0/1.0 (적 약화 보정 제거), Hard 1.7/1.45, Nightmare 2.9/2.0.
- 시간 스케일 분당 HP +34% / DMG +20% / SPD +8%로 상향 — 후반 압박 강화.
- 피격 무적 0.5s → 0.35s, 무기 레벨업 곡선 DMG ×1.22 / CD ×0.90 (스노볼 완화).
- 측정 하니스 `tests/sim_afk_run.gd` 추가 (무입력/단순도주 봇, time_scale 가속).

### Changed — 하위 화면 UI 통일 (UIKit)

- `UIKit.gd` 공용 크롬 신설: 밤하늘 백드롭 + 기억의 금색 타이틀 + 석판 카드 + 선택 카드 금색 글로우 프레임 + 단일 텍스트 팔레트.
- 상점: 무지개색 텍스트 정리, 석판 카드 + 금색 비용/레벨 + 석판 구매 버튼.
- 캐릭터/스테이지 선택: 석판 카드 + 선택 글로우 + 석판 액션 버튼. 설정: 달빛 슬라이더.

### Fixed

- 레벨업 카드의 등급 글로우 프레임이 출시 이후 한 번도 렌더링되지 않던 버그 수정 (Godot 4.2에 없는 API 호출로 매번 스킵).
- 카드 텍스트가 테두리 장식·글로우 프레임과 겹치던 증상 수정 — 석판 텍스처의 투명 거터 크롭(128×160→96×160) + 글로우를 PanelContainer 자식 노드에서 draw 시그널 직접 드로잉으로 교체 (자식 Control이 콘텐츠 영역에 강제 배치되는 레이아웃 함정 회피).

### Verification

- `tests/verify_ui_wiring.gd` headless ALL OK, 시뮬 런 스크립트 에러 0.
- 봇 측정: 단순 도주 봇 Normal 무한 생존 가능 → 73초 사망, Hard 57초 사망.
- 상점/캐릭터/스테이지/설정 720p 캡처로 겹침 해소·테마 통일 확인.
