## 🌙 v0.15.0 — 한 판이 짧아지고 부드러워졌다

한 판 길이를 **10분 → 7분**으로 줄이고, Normal 난이도를 살짝 풀어 입문 장벽을 낮춤. 죽었을 때 "다시 하기 싫음" 비용이 줄어들고, 출퇴근 한 정거장에 한 판이 끝납니다.

### ⏱️ 게임 길이 단축

- **Forest / Frozen Wastes / Twilight Sanctum / Inferno Chasm**: 600초(10분) → **420초(7분)**
- **Cursed Tomb** (최종 스테이지): 720초(12분) → **480초(8분)**
- 모든 wave 타임라인을 비례 압축 — 후반 러시 구간도 그만큼 빨리 도달
- 보스 등장: 마지막 30초 그대로 유지 (Forest 기준 6분 30초 → 보스)
- **미니보스 등장**: 120/240/360/480초 → **80/160/240/320초** 로 당김

### 🟢 Normal 난이도 풀기

- 적 **HP 배율 1.0 → 0.85** (15% 약화)
- 데미지·골드 보상은 그대로 유지 → "더 빨리 죽는데 보상은 같음"
- **Hard / Nightmare는 그대로** — 도전 모드는 보존
- 추가로 7분에서 끝나니까 시간 기반 적 강화(분당 HP +20%)도 자연스럽게 약해짐 (보스 시점 HP 배수 ~2.9x → ~2.3x)

### 🏃 곁들여 조정한 것

- **speed_runner 업적**: 10레벨까지 180초 → **126초** (게임 길이 비율 유지)
