## v0.34.1 — 스토리 화면 스크롤 안정화

### 핵심 변경

Story Chronicle 화면에서 긴 장문 카드를 읽을 때 모바일 터치 드래그가 카드 내부 컨트롤에 막히지 않도록 스크롤 입력 처리를 보강했습니다.

### Fixed

- 스토리 카드 내부의 읽기 전용 `PanelContainer` / `Label` / 장식 컨트롤이 터치 입력을 독점하지 않고 `ScrollContainer`까지 전달되도록 조정했습니다.
- 세로 스크롤바를 항상 표시해, 스토리 화면이 아래로 더 이어진다는 시각적 단서를 강화했습니다.
- 가로 스크롤은 비활성화해 세로 장문 읽기 동작에 집중하도록 정리했습니다.

### Verification

- Godot headless 풀 프로젝트 로드: 스크립트 에러 없음. 종료 시 기존과 같은 ObjectDB leak 경고로 exit code 1 반환.
- StoryUI 단독 로드: 스크립트 에러 없음. 종료 시 기존과 같은 ObjectDB leak 경고로 exit code 1 반환.
