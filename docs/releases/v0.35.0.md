## v0.35.0 — 전투 UI 폴리시와 앱 용량 다이어트

### 핵심 변경

생성만 해두고 연결하지 않았던 AI 픽셀아트 UI 자산을 결과 화면·HUD·조이스틱·메인메뉴에 일괄 배선하고 (UI 로드맵 Phase UI-5 완료), 표시 크기 대비 최대 15배 오버스펙이던 스토리/배경 PNG를 다운스케일해 자산 용량을 13.4MB → 2.3MB로 줄였습니다.

### Added — UI 자산 연결

- 결과 화면: 승리 시 트로피 현수막(`banner_stage_clear`) 안에 타이틀 표시, 석판 패널(`panel_card_dark.9`) 적용, 획득 골드를 코인 아이콘 + 금색 전용 행으로 강조. 패배는 기존 붉은 타이틀 유지로 승/패가 한눈에 구분됩니다.
- HUD: 남은 시간 옆 시계 아이콘, 처치 카운트를 임시 단검 → 전용 해골 아이콘으로 교체.
- 가상 조이스틱: 코드 원 드로잉 → 달빛 링/썸 텍스처 (미임포트 환경에서는 기존 절차적 드로잉으로 fallback).
- 메인메뉴: 로고 뒤 달빛 덩굴 장식(`bg_logo_glow_ornament`), 설정 버튼에 기어 아이콘.

### Changed — 자산 최적화

- 스토리 인장 5종·책 아이콘: 1254×1254 → 256×256 (표시 크기 최대 80px의 3배 헤드룸).
- 봉인 체인: 2508×627 → 1024×256. 풀스크린 배경 3종은 치수 유지 + 256색 팔레트 양자화.
- 레거시 `shop_warriors_might.png` 삭제 (미참조).
- `docs/ASSET_GUIDE.md` 현행화 — "표시 크기 3배 초과 원본 커밋 금지" 규율 박제.

### Verification

- `tests/verify_ui_wiring.gd` headless 검증 ALL OK (씬 로드 + @onready 노드 경로 + 텍스처 로드).
- `tests/screenshot_result_panel.gd`로 승리/패배 결과 화면 캡처 확인. 메인메뉴/인게임 HUD 윈도우 캡처 확인.
- 다운스케일 자산 시각 품질 열화 없음 (캡처 대조).
