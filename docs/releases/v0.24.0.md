## 🌙 v0.24.0 — 메인 메뉴 Nightseed 리워크 1차 + AdMob 보상형 SDK

### 🎯 핵심 변경

메인 메뉴를 단색 ColorRect + 사각 버튼 모음에서 **"달빛 밤숲 앞에 선 캐릭터 + 달빛 PLAY CTA + 석판 보조 메뉴"** 구성으로 전환. 새 이미지 애셋 없이 절차 묘사와 기존 16×16 캐릭터 스프라이트만으로 1차 비주얼 톤을 잡았다. AdMob 보상형 광고 SDK도 함께 통합(테스트 ID 단계)되어 실제 광고 ID만 받으면 라이브 전환 가능.

---

### 🌑 메인 메뉴 — Nightseed 톤으로

`docs/UI_ART_DIRECTION_ROADMAP.md`의 Phase UI-1 + UI-2 1차 적용.

- **절차 야경 배경 (`MenuBackdrop.gd`)** — 깊은 남색 → 숲 녹색 32단 수직 그라데이션, 별 80개, 우상단 달 + 6단 푸른 헤일로, 안개 가로 띠, 앞/뒤 2단 트리 실루엣, 호박색 반딧불 16개. 모두 코드 _draw로만 그림 — 라이선스 부담 0
- **캐릭터 쇼케이스 (`CharacterShowcase.gd`)** — 현재 선택 캐릭터 스프라이트(16×16)를 NEAREST 필터로 6× 업스케일 + 달빛 후광 6단 + 봉인 링 2줄 + 이름 라벨. 스프라이트 누락 시 silhouette fallback. 캐릭터 변경하고 돌아오면 즉시 갱신
- **버튼 위계 재정리 (`ButtonStyles.gd` 확장)**
  - PLAY → **달빛 CTA** (창백한 달빛 배경 `#DDEBFF` + 짙은 남색 텍스트 + 외곽 글로우)
  - 1차 행 (CHARACTERS / STAGES / DIFFICULTY) → **강조색 테두리 석판**
  - 2차 행 (SHOP / STORY / LEADERS) → **조용한 석판**
  - 상단 우측 코너 (Language / Credits) → 더 작은 석판 보조
  - 모서리 반경 ≤6, 테두리 2~3px — 봉인석/석판 느낌
- **타이틀/부제** 짙은 남색 외곽선 추가 — 배경 위에서 떠 보이도록
- **상태 카드** 더 어두운 청회색 + 상단 룬 라인 + 모서리 반경 12→6

### 📺 AdMob 보상형 광고 SDK 통합

Poing Studios `godot-admob-plugin v4.3.1` + Android backend v4.2.0 (ads only) 배치.

- 부활 / 골드 2배 CTA의 백엔드를 새 SDK API로 재작성 (`AdManager.gd`)
- 외부 시그널/메서드 인터페이스(`rewarded_granted/dismissed/failed`, `show_rewarded(tag)`) 유지 — GameRoot 호출부 변경 없음
- `ENABLED=true`로 **Google 공식 테스트 광고 단위 ID** 사용 — 폰에서 "Test Ad" 워터마크가 보이는 진짜 테스트 광고가 재생됨
- 실제 광고 ID 수급(Play Console 공개 트랙 출시 후 AdMob 콘솔 등록)되면 두 상수만 교체
- R8 keep 룰을 `proguard-rules.pro`에 추가
